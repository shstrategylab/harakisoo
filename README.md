# 하락이수 (河洛理數) — AI 자동 풀이 서비스

생년월일시를 입력하면 사주팔자 → 수리 변환 → 64괘 자동 도출까지
모든 계산을 서버에서 처리하고, Claude 심층 상담용 프롬프트를 자동 생성합니다.

## 🚀 설치 및 실행

### 1. 의존성 설치
```bash
cd harakisu
npm install
```

### 2. 개발 서버 실행
```bash
npm run dev
# http://localhost:3000
```

### 3. 프로덕션 빌드
```bash
npm run build
npm start
```

## ⚠️ 보완 필요 사항 (개발 전 확인)

### 1. manse_100_db.json — daily_cache 범위 확장 필요
- 현재: 2023~2027년(5년치)만 있음
- 문제: 1950~2022년 출생자의 일주(日柱)는 60갑자 역산으로 처리됨
- 해결: 정확도를 높이려면 1950~2022년 일진 데이터 추가 필요
- **현재 코드**: 기준점(2024-01-01=甲子)에서 역산하므로 동작은 하나,
  실제 만세력과 1~2일 오차 가능성 있음 → 검증 필요

### 2. solar_terms_db.json — 2010년 이전 출생자 월주 계산
- 현재: 2010~2050년 절기 데이터만 있음
- 문제: 1930~2009년 출생자의 월주는 절기 기준으로 정확히 못 잡을 수 있음
- 해결: 과거 절기 데이터 추가 or 별도 계산 로직 작성 필요

### 3. 하락이수 수리표 검증
- `harakisu_core.js`의 `CHUNGAN_SU`, `JIJI_SU` 값을 보유하신 원문과 대조 확인 필요
- 유파에 따라 수리 값이 다를 수 있음

### 4. 성별에 따른 괘 도출 방향 확인
- 현재: 남성=천수→상괘/지수→하괘, 여성=반대
- 원문 또는 사용하실 유파의 규칙과 일치하는지 확인 필요

## 📄 페이지별 기능

| 페이지 | 경로 | 설명 |
|--------|------|------|
| 랜딩 | `/` | 서비스 소개, 시작하기 |
| 입력 | `/input` | 4단계 폼 (성별→생년월일→시각→확인) |
| 결과 | `/result` | 6탭 (개요/선천괘/후천괘/유년괘/타임라인/수리) |
| 64괘 사전 | `/gua-dictionary` | 괘 검색·클릭 상세 + 6효 전체 |
| 로드맵 | `/guide` | 4단계 학습 가이드 |
