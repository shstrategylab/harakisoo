# 하락이수 (河洛理數)
> 주역으로 읽는 나의 인생 지도

생년월일시를 입력하면 사주팔자 → 수리 변환 → 64괘 자동 도출까지  
모든 계산을 브라우저에서 처리하고, Claude 심층 상담용 프롬프트를 자동 생성합니다.

🔗 **서비스 주소:** https://shstrategylab.github.io/harakisoo/

---

## 🧮 계산 흐름

```
생년월일시 입력
    ↓
만세력 계산 (harakisu.js)
  양력 → 연주·월주·일주·시주 (사주팔자 8글자)
    ↓
수리 변환
  천간수·지지수 → 천수(홀수 합) · 지수(짝수 합)
    ↓
괘 도출
  선천괘 (천수÷8, 지수÷8 나머지 → 상괘·하괘)
  원당효 (천수+지수 ÷6 나머지)
  후천괘 (원당효 변효)
  유년괘 (현재 나이 기준 대운 구간)
    ↓
DB 조회 (data/*.json fetch)
  괘사·효사·현대해석 매칭
    ↓
결과 출력 + Claude 프롬프트 자동 생성
```

## 🤖 Claude 프롬프트 생성기

풀이 결과 페이지 하단에서 자동으로 조립됩니다.

1. 추가 질문 입력 (선택)
2. **프롬프트 복사** 버튼 클릭
3. [claude.ai](https://claude.ai) 대화창에 붙여넣기

생성되는 프롬프트에는 선천괘·후천괘·원당효사·유년괘·현재 대운 정보가 모두 포함됩니다.

## ⚠️ 보완이 필요한 사항

### 1. 일주(日柱) 정확도
- `manse_100_db.json`의 `daily_cache`가 2023~2027년(5년치)만 수록
- 그 외 연도는 기준점(2024-01-01 = 甲子)에서 **60갑자 역산**으로 처리
- 실제 만세력과 **1~2일 오차** 가능성 있음 → 검증 후 데이터 보완 필요

### 2. 월주(月柱) 정확도
- 절기 DB가 2010년부터 수록되어 있어 **2009년 이전 출생자**의 월주 계산 시 절기 기준이 부정확할 수 있음
- 과거 절기 데이터 추가 또는 계산 로직 보완 필요

### 3. 수리표 검증
- `harakisu.js`의 `CHUNGAN_SU`, `JIJI_SU` 값을 보유하신 원문과 대조 확인 필요
- 유파에 따라 수리 값이 다를 수 있음

### 4. 성별 방향 확인
- 현재: **남성** = 천수→상괘 / 지수→하괘, **여성** = 반대로 적용
- 사용하실 유파의 규칙과 일치하는지 확인 필요
